Polymer Selection Logic

Melt Index (MI)

  • Low MI (0.05–1) → High strength → Pipes, films
  • Medium MI (1–10) → Balanced → Blow molding
  • High MI (>10) → Easy flow → Injection molding

Density

  • Higher density → Higher stiffness
  • Lower density → Better flexibility

Molecular Weight Distribution

  • Broad MWD → Better processing window
  • Narrow MWD → Uniform performance

Application Logic

  • Film → Toughness + sealability
  • Injection → Flow + dimensional stability
  • Pipe → Strength + pressure resistance
  • Blow molding → ESCR + stiffness

UV Selection Rule

  • Use "U" grades for outdoor applications

Source: Derived from GAIL grade slate and application data
